Salles informatiques

Le département Informatique dispose de six salles équipées accessibles aux étudiants pendant les heures d'ouverture du campus. Elles sont principalement réservées aux TD et TP encadrés, mais certaines sont en libre accès hors enseignement.

Capacité : 30 postes
Configuration : Intel Core i7, 16 Go RAM, Ubuntu 22.04 LTS + Windows 11 dual-boot
Usage : TD/TP — réservée aux groupes planifiés
Accès libre : Non

Salle A102

Capacité : 28 postes
Configuration : Intel Core i7, 16 Go RAM, Windows 11
Usage : TD/TP spécialisé — développement web & BDD
Accès libre : Non

Salle A103

Capacité : 24 postes
Configuration : Intel Core i9, 32 Go RAM, Kali Linux + Windows 11
Usage : Lab cybersécurité — accès restreint, sur autorisation
Accès libre : Sur demande enseignant

Salle B204

Capacité : 32 postes
Configuration : Intel Core i5, 8 Go RAM, Windows 11
Usage : Polyvalente — libre accès hors créneaux réservés

Salle B205

Capacité : 20 postes
Configuration : Intel Core i7, 16 Go RAM, Ubuntu 22.04 LTS
Usage : Projets étudiants — libre accès en soirée
Accès libre : Oui (8h–22h)

Capacité : 16 postes + bancs électroniques
Configuration : Raspberry Pi 4, Arduino, oscilloscopes, alimentation réglable
Usage : Systèmes embarqués — sur réservation
Accès libre : Sur inscription via formulaire

Comment obtenir du support technique

1

Consultez la FAQ en ligne

La majorité des problèmes courants (connexion au Wi-Fi, activation de licence, accès VPN, oubli de mot de passe) sont documentés sur le portail d'aide étudiant. Commencez par là avant d'ouvrir un ticket.

2

Ouvrez un ticket via le portail

Si la FAQ ne répond pas à votre problème, créez un ticket sur le portail Helpdesk avec votre adresse @efrei.net. Précisez le matériel concerné, le système d'exploitation et le message d'erreur exact. Plus votre description est précise, plus la résolution sera rapide.

3

Passez en présentiel au bureau DSI

Pour les problèmes nécessitant une intervention physique (poste bloqué, carte réseau, matériel endommagé), passez au bureau de la Direction des Systèmes d'Information situé au bâtiment C, bureau C015, rez-de-chaussée. Un technicien est disponible sans rendez-vous pendant les heures d'ouverture du support.

4

Contact d'urgence

En cas d'incident bloquant en plein TD ou TP (panne de serveur, coupure réseau en salle), contactez directement l'enseignant qui peut activer la procédure d'escalade vers l'équipe technique en moins de 15 minutes. L'adresse email de support est disponible dans le portail.

Réseau Wi-Fi & accès à distance

Wi-Fi campus

Le réseau EFREI-Student est accessible sur l'ensemble du campus avec vos identifiants EFREI. Une deuxième connexion est possible via Eduroam, qui donne accès au Wi-Fi dans toutes les universités partenaires en France et en Europe.

  • Identifiants : adresse @efrei.net + mot de passe portail
  • Bande passante : 1 Gbps fibre — partagée par salle
  • Couverture : 100 % des bâtiments A, B, C

Accès VPN & ressources distantes

Depuis chez vous, le VPN institutionnel permet d'accéder aux serveurs de développement, aux licences réseau (MATLAB, Visual Studio) et aux machines virtuelles du département comme si vous étiez sur le campus.

  • Client VPN : Cisco AnyConnect (téléchargement sur le portail)
  • Authentification double facteur requise
  • Disponible 24h/24, 7j/7
