A propos de ce site

Site realise dans le cadre du projet TI402 — classe P2 SC4, promo 2025-2026.

Le projet TI402

Ce site a ete fait dans le cadre du module TI402 de M. Hamidi. L'objectif c'etait de creer un site vitrine pour le departement info de l'EFREI, uniquement en HTML, CSS et JavaScript — sans framework.

On a essaye de couvrir un maximum de choses : les formations, les enseignants, la vie etudiante, les partenariats, un systeme de permanences et une page contact. Ca fait beaucoup de pages mais ca nous a permis de mettre en pratique tout ce qu'on a vu en cours.

Pour le design on a suivi la charte graphique de l'EFREI. Le bleu principal c'est le #0A316C, on a utilise Montserrat pour les titres et Open Sans pour le texte normal. On aurait voulu faire quelques trucs en plus mais le temps manquait.

Technologies

HTML5 pour la structure, CSS3 pour le design (avec les media queries pour le responsive), JavaScript pour l'interactivite — le carousel, la validation des formulaires, le systeme de permanences avec localStorage, etc. Aucune bibliotheque externe utilisee.
