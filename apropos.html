À propos de ce site

Ce site a été réalisé dans le cadre du projet de programmation web TI402 — classe P2 SC4, promotion 2025-2026.

Le projet TI402

Ce site a été réalisé dans le cadre du module TI402 encadré par M. Hamidi. L'objectif était de concevoir un site vitrine pour le département informatique de l'EFREI, uniquement en HTML, CSS et JavaScript — sans utiliser de framework.

On a voulu faire quelque chose de complet avec plusieurs pages : les formations, les enseignants, la vie étudiante, les partenariats, un système de permanences et une page contact. C'était pas mal de travail mais ça nous a permis de vraiment pratiquer ce qu'on a vu en cours.

Pour le design on s'est basés sur la charte graphique fournie avec le bleu EFREI, Montserrat pour les titres et Open Sans pour le texte.
